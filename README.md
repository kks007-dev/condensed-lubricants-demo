# Lubricants Voice Demo

An AI-assisted order flow demonstration with keyboard-driven navigation and animated transitions.


## Prerequisites

- Node.js 18+ 
- npm or yarn

## Installation

1. Install dependencies:
```bash
npm install
```

## Development

Start the development server:
```bash
npm run dev
```


## Demo Controls

Once the app is running, you can navigate through the demo steps using:

- **Next Step**: Press `Space` or `→` (right arrow)
- **Previous Step**: Press `←` (left arrow)  
- **Restart**: Press `R`


## Demo Flow

The demo showcases an AI-assisted order processing workflow:

1. **Initial View** - Draft order with capacity issues
2. **Capacity Warning** - Order exceeds container limits  
3. **AI Suggestions** - Alternative configurations offered
4. **Order Updated** - AI applies recommended changes
5. **Freight Setup** - Scheduling with BlueBridge Freight
6. **Confirmation** - Pickup confirmed and email sent
7. **Contingency** - Product substitution preferences
8. **Final Confirmation** - All updates verified
